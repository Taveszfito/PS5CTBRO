## ✨ Features

### 🎮 Input & Interaction

* Full input test (USB / Bluetooth)
* Touchpad input with visual feedback (USB / Bluetooth)
* Controller remains usable for Android games while advanced controller features are active (USB)

### 🎯 Adaptive Triggers

* Precise resistance and vibration control (USB)

### 🔊 Audio

* Live Android audio routed to the controller through a custom audio pipeline (USB)
* Controller speaker playback (USB)
* Stereo audio routing to the controller headphone jack (USB)
* Dedicated controller volume control 
* Integrated Android volume panel support 

### 🎤 Microphone

* Controller microphone testing (USB)
* Record audio from the DualSense microphone
* Play back recorded audio through the phone speaker or the controller speaker

### 📳 Haptics / Vibration

* Standalone vibration testing (USB / Bluetooth)
* Audio-reactive vibration (USB)
* Experimental Game Mode that converts captured game audio into DualSense haptic feedback (USB)

### 🌈 LED Control

* Light bar and player LED control (USB)
* Audio-reactive "Disco Mode" (USB)

### 🌀 Motion Sensors

* Real-time gyroscope testing (USB)
* Live motion visualization (USB)

### 🛠️ Advanced Tools

* Hidden Byte Debug screen for sending custom DualSense reports and hex commands (USB)
* Send custom hex flags
* Send full HID reports
* Save and replay custom byte sequences
* Unlock by tapping the app version number 7 times

### ℹ️ Controller Info

- Dedicated controller information screen (USB)
- Displays controller information and battery level (USB)
